# 🥗 냉장고 파먹기(냉파) 도우미

"버려지는 식재료 없이, 알뜰한 자취 생활의 시작!"

## 📊 프로젝트 한눈에 보기
이 프로그램은 식재료의 유통기한을 관리하여 낭비를 줄이는 **자취생 필수 도우미**입니다.

| 기능 분류 | 핵심 내용 | 비고 |
| :--- | :--- | :--- |
| **식재료 관리** | 등록, 수정, 삭제(CRUD) | 간편한 입력폼 |
| **유통기한 모니터링** | 임박(3일 이내) 식재료 강조 | 시각적 알림 |
| **정렬 알고리즘** | 유통기한 순 자동 정렬 | 우선순위 자동화 |

---

## 🛠️ 기술 스택 (Tech Stack)
> 본 프로젝트는 복잡한 서버 없이 브라우저 환경에서 즉시 구동됩니다.

* **Frontend:** `HTML5`, `CSS3`
* **Logic:** `Vanilla JavaScript` (외부 라이브러리 미사용)
* **Deployment:** `GitHub Pages`

---

## 🚀 어떻게 실행하나요?
프로그램을 실행하려면 아래 순서를 따라주세요.

1. **저장소 클론:** `git clone [주소]`
2. **파일 열기:** 폴더 내의 `index.html` 파일을 크롬(Chrome) 또는 엣지(Edge) 브라우저로 엽니다.
3. **사용 시작:** 이름을 입력하고 날짜를 선택하여 '추가하기' 버튼을 누릅니다.

```javascript
// 유통기한 임박 계산 로직 예시
if (diffDays <= 3) {
    // 3일 이내면 '임박' 클래스 부여하여 시각화
    li.classList.add('urgent');
}
